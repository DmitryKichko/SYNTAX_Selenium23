Selenium Class 2
=========================================================================================================

Что такое веб-элемент?

Все, что мы видим на веб-странице, считается веб-элементом
например, кнопки, текстовые поля, чек-боксы, ссылки.

HTML - HyperText Markup Language (Язык разметки гипертекста)

Что такое DOM?
DOM расшифровывается как Document Object Model. Это внутреннее представление веб-страницы.

Также называется инструментами разработчика или Dev tools.
Документ --> может быть сохранен любой документ, например, pdf, excel.
Объект --> любая строка в HTML рассматривается как объект
Модель --> это просто макет HTML.

Локатор
это адрес или позиция веб-элемента на веб-странице
используется для определения местоположения или идентификации веб-элементов на веб-странице

8 локаторов:
- ID
- Name
- Tagname
- xpath
- linktext
- partialLinkText
- CSS Selector
- ClassName

Значение сценария HTML:
фиолетовый - теги
коричневый -- атрибуты
синий -- значения атрибутов


Почему нам нужно много локаторов?
-- некоторые локаторы могут иметь дублирующие значения в других узлах.
-- разработчики могут не предоставить все локаторы для всех элементов
-- некоторые локаторы могут быть неприменимы для искомых элементов.
    например, LinkText и partiaLinkText используются только для ссылок.

тег "a" также называется тегом якоря

если 2 или более элементов имеют одинаковые атрибуты и значения атрибутов, Selenium будет взаимодействовать только с
первым элементом, который он обнаружит
